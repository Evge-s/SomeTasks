# SoftServe .Net C# Certification Tasks

1. Два масиви перевірити чи 2й массив є реверсом першого.

2. Дано число 13345, зробити з нього string 1, 3, 3, 4, 5.

2. Дано число 3542, зробити з нього string 3000, 500, 40, 2.

3. Дано массив з числами від 0 до скількись там..
   треба визначити якої цифри по порядку не вистачає в послідовності, наприклад у массиві 0,1,2,3,5 не вистачає 4, тоді треба повернути 4.

4. Знайти певну цифру в масиві, якщо вона є то вивести її, якщо ні то -1.

5. Дано 2 массива, повернути 3й в якому по черзі поскладані значення з 2х вхідних массивів

6. Дано List<string> емейлів, повернути кількість емейлів які повторюються
   наприклад є List<string> emails
   у ньому:
   email_1@gmail.com<br>
   email_1@gmail.com<br>
   email_2@gmail.com<br>
   email_3@gmail.com<br>
   email_3@gmail.com<br>
   треба повернути кількість емейлів які повторяються (2)

7. Повернути найдовше слово в рядку.

8. Привести число до вказаного степеня і повернути результат.

9. Дано два string. Де один довгий, а інший на пару символів з пробілами. При наявності другого в першому потрібно вивести індекс із кінця строки.

10. Дано клас. Змінити його так, щоб його можна було використовувати у якості ключа словника.

11. Дано string типу 11+34+5. Тільки цифри і плюси. Вивести результат обчислення.

12. Дано текст, який може містити коми, крапки і пробіли, крім слів. Вивести найдовше слово.

13. Дано текст, який може містити коми, крапки і пробіли, крім слів. Вивести список унікальних слів.

14. Дано масив інтів [1,2,3,5,5]. Повернути Dictionary де кей - число у массиві, а value - кількість його повторень у массиві.
